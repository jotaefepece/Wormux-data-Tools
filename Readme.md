# 🐛 Wormux Data Tools 🐛 

Small collection of TCP tools for testing, experimenting… and a bit of chaos.

---

## Tools

- **XPaTCPfull**  
  TCP 1-to-1 communication

- **dataXYaTCP**  
  Send data to multiple clients (1-to-many)

- **WormuxTCP**  
  Random messages + random timing per client
