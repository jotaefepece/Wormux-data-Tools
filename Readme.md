# 🐛Wormux Data Tools🐛 

Colección de herramientas TCP para pruebas, experimentación… 
y un poco de caos.

---

## Herramientas

- **XPaTCPfull**  
  Comunicación TCP 1 a 1

- **dataXYaTCP**  
  Envío de datos a múltiples clientes (1 a muchos)

- **WormuxTCP**  
  Envío de mensajes aleatorios con tiempos aleatorios por cliente
